# MS Learn Navigator

A browser extension that enhances Microsoft Learn with a persistent sidebar TOC, progress tracking, and resume navigation.

## Features

- 📚 Expand-to-top Dynamic Table of Contents (TOC) sidebar
- ✅ Module completion tracking (Summary-based)
- 🔁 Resume where you left off
- 🧭 Learning path awareness and navigation
- 🎯 Expand-to-top UX for focused learning

## Install

1. Clone repo
2. Go to chrome://extensions
3. Enable Developer Mode
4. Load unpacked

## Usage

Load the extension in Chrome (Developer Mode) and open any Microsoft Learn course:
https://learn.microsoft.com/training/
